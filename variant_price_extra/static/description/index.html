Product Variant's Price Extra

Set and Manage different Extra Price for a Product Variant

This module allows you to set and manage extra price for each product's variant separately.

You can simply set the extra price to be applied on a product's variant, regardless of it's attribute and attribute value.

Extra Price is applicable throughout Odoo

Point Of Sale

Extra price is automatically applied in Point of Sale. No need of any manual intervention.

Website Sale

Extra price for Product's Variant applicable in Website sale too.
